                                  *Game Rules*

You're prisoner in "Graduation Prison" which has nine strong walls to be 
Broken, your only weapon is tank but unfortunately without bombs, so you've
To answer each level question which is equivilant to each phase in education
Journey untill graduation, so you've to gain bullets and make holes through
walls to survive. But if your answer is wrong, you'll be caught and the game is over.

                                  *Controls*
-To move, Use arrows.
-To fire a bullet in walls, Press 'F'.
-To exit, Press 'Esc'.
-To answer a question,Press on numbers in keyboard(number is only one digit for any question).
